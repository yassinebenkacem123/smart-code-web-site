Introduction au CSS

Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire la présentation d'un document HTML. Il permet de contrôler l'apparence des éléments d'une page web.

Le CSS sépare le contenu (HTML) de la présentation (styles), ce qui améliore la maintenabilité des sites web.

Syntaxe CSS de base

/* Sélecteur */
sélecteur {
  /* Propriété : Valeur; */
  propriété: valeur;
}

Méthodes d'intégration du CSS

  • CSS interne : Utilisation de la balise <style> dans le <head>
  • CSS externe : Liaison avec un fichier .css via <link>
  • CSS en ligne : Utilisation de l'attribut style sur les éléments HTML

Sélecteurs CSS

Les sélecteurs permettent de cibler les éléments HTML à styliser :

/* Sélecteur d'élément */
p { color: blue; }

/* Sélecteur de classe */
.ma-classe { font-size: 16px; }

/* Sélecteur d'identifiant */
#mon-id { background: yellow; }

/* Sélecteur d'attribut */
a[target="_blank"] { color: red; }

/* Sélecteurs combinés */
div p { margin: 10px; } /* Descendant */
div > p { padding: 5px; } /* Enfant direct */
h1 + p { font-weight: bold; } /* Adjacent */
h2 ~ p { font-style: italic; } /* Frère */

Propriétés CSS essentielles

Couleurs et arrière-plans

body {
  color: #FFFFFF; /* Couleur du texte */
  background-color: #000000; /* Couleur d'arrière-plan */
  background-image: url('image.jpg'); /* Image de fond */
  background-size: cover; /* Taille de l'image */
  background-position: center; /* Position de l'image */
}

Texte et polices

p {
  font-family: Arial, sans-serif; /* Police */
  font-size: 16px; /* Taille */
  font-weight: bold; /* Gras */
  text-align: center; /* Alignement */
  line-height: 1.6; /* Hauteur de ligne */
  text-decoration: underline; /* Soulignement */
}

Modèle de boîte (Box Model)

Chaque élément HTML est représenté comme une boîte composée de :

  • Contenu (content)
  • Remplissage (padding)
  • Bordure (border)
  • Marge (margin)
div {
  width: 300px; /* Largeur du contenu */
  height: 200px; /* Hauteur du contenu */
  padding: 20px; /* Remplissage intérieur */
  border: 5px solid blue; /* Bordure */
  margin: 30px; /* Marge extérieure */
  box-sizing: border-box; /* Modèle alternatif */
}

Disposition et mise en page

Flexbox

Module de mise en page unidimensionnel pour répartir l'espace :

.conteneur {
  display: flex;
  flex-direction: row; /* row, column */
  justify-content: center; /* Alignement horizontal */
  align-items: center; /* Alignement vertical */
  flex-wrap: wrap; /* Passage à la ligne */
}

.element {
  flex: 1; /* Répartition de l'espace */
}

CSS Grid

Système de mise en page bidimensionnel :

.conteneur {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 colonnes égales */
  grid-template-rows: 100px auto;
  gap: 20px; /* Espace entre les éléments */
}

.element {
  grid-column: span 2; /* Occupe 2 colonnes */
  grid-row: 1; /* Position sur la 1ère ligne */
}

Positionnement

Contrôle de la position des éléments :

.element {
  position: static; /* Par défaut */
  position: relative; /* Position relative */
  position: absolute; /* Position absolue */
  position: fixed; /* Position fixe */
  position: sticky; /* Position collante */
  top: 10px;
  right: 20px;
  z-index: 10; /* Ordre d'empilement */
}

Effets avancés et responsivité

Transitions et animations

/* Transition simple */
.bouton {
  transition: background-color 0.3s ease;
}
.bouton:hover {
  background-color: blue;
}

/* Animation avec keyframes */
@keyframes example {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}
.element {
  animation: example 2s infinite;
}

Design Responsive

Adapter le design à différentes tailles d'écran :

/* Media queries */
@media screen and (max-width: 768px) {
  .conteneur {
    flex-direction: column;
  }
  .element {
    width: 100%;
  }
}

/* Unités responsives */
.element {
  width: 100%; /* Pourcentage */
  padding: 2vw; /* Viewport Width */
  font-size: clamp(16px, 3vw, 24px); /* Taille fluide */
}
