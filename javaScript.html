Introduction à JavaScript

Qu'est-ce que JavaScript ?

JavaScript est un langage de programmation qui permet de rendre les pages web interactives. Contrairement au HTML (structure) et au CSS (style), JavaScript ajoute des fonctionnalités dynamiques.

JavaScript est le seul langage de programmation qui fonctionne nativement dans tous les navigateurs web modernes.

Intégration de JavaScript

Il existe plusieurs façons d'intégrer JavaScript dans une page HTML :

<!-- Méthode 1 : Dans le head -->
<script src="script.js"></script>

<!-- Méthode 2 : À la fin du body -->
<script>
  // Votre code JavaScript ici
  console.log("Bonjour JavaScript !");
</script>

Variables et types de données

JavaScript utilise trois mots-clés pour déclarer des variables :

// Déclaration de variables
let age = 25; // Modifiable
const PI = 3.14; // Constante
var ancienneVariable = "déconseillée"; // Ancienne méthode

// Types de données
let nom = "Alice"; // String (chaîne de caractères)
let estEtudiant = true; // Boolean (booléen)
let notes = [15, 18, 12]; // Array (tableau)
let personne = { // Object (objet)
  nom: "Bob",
  age: 30
};

Fonctions et structures de contrôle

Déclaration de fonctions

Les fonctions permettent de regrouper du code réutilisable :

// Déclaration de fonction
function saluer(nom) {
  return "Bonjour " + nom + " !";
}

// Fonction fléchée (ES6)
const additionner = (a, b) => {
  return a + b;
};

// Appel de fonction
console.log(saluer("Alice")); // "Bonjour Alice !"
console.log(additionner(5, 3)); // 8

Conditions (if...else)

let temperature = 25;

if (temperature > 30) {
  console.log("Il fait très chaud !");
} else if (temperature > 20) {
  console.log("Il fait bon.");
} else {
  console.log("Il fait frais.");
}

Boucles (for, while)

// Boucle for
for (let i = 0; i < 5; i++) {
  console.log("Itération " + i);
}

// Boucle while
let compteur = 0;
while (compteur < 3) {
  console.log("Compteur: " + compteur);
  compteur++;
}

// Boucle for...of (pour les tableaux)
const fruits = ["pomme", "banane", "orange"];
for (const fruit of fruits) {
  console.log(fruit);
}

Manipulation du DOM

Qu'est-ce que le DOM ?

Le DOM (Document Object Model) est une représentation en mémoire de la page web. JavaScript peut modifier le DOM pour changer le contenu, la structure et le style d'une page.

Sélection d'éléments

// Sélection par ID
const titre = document.getElementById("monTitre");

// Sélection par classe
const elements = document.getElementsByClassName("maClasse");

// Sélection par sélecteur CSS
const premierPara = document.querySelector("p");
const tousLesPara = document.querySelectorAll("p");

Modification du contenu

Ceci est un texte de démonstration.

// Changer le contenu textuel
function changerTexte() {
  document.getElementById("demoText").textContent = "Texte changé avec JavaScript !";
}

// Ajouter un nouvel élément
function ajouterElement() {
  const nouvelElement = document.createElement("p");
  nouvelElement.textContent = "Nouvel élément ajouté !";
  document.getElementById("demoContainer").appendChild(nouvelElement);
}

// Méthode 1 : Dans le HTML

Manipulation de tableaux

const nombres = [1, 2, 3, 4, 5];

const doubles = nombres.map(n => n * 2); // [2, 4, 6, 8, 10]

Promesses et async/await

Les promesses permettent de gérer les opérations asynchrones :

// Création d'une promesse
const maPromesse = new Promise((resolve, reject) => {
  setTimeout(() => {
    const succes = true;
    if (succes) {
      resolve("Opération réussie !");
    } else {
      reject("Échec de l'opération");
    }
  }, 2000);
});

// Utilisation avec .then() et .catch()
maPromesse
  .then(resultat => console.log(resultat))
  .catch(erreur => console.error(erreur));

// Utilisation avec async/await
async function executerPromesse() {
  try {
    const resultat = await maPromesse;
    console.log(resultat);
  } catch (erreur) {
    console.error(erreur);
  }
}

Manipulation des dates

// Créer une date
const maintenant = new Date();

// Obtenir des composants de date
const jour = maintenant.getDate();
const mois = maintenant.getMonth() + 1; // Les mois commencent à 0
const annee = maintenant.getFullYear();

// Formater une date
function afficherDate() {
  const dateStr = `${jour}/${mois}/${annee}`;
  document.getElementById("dateResult").textContent = dateStr;
}
