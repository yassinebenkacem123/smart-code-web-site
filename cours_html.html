Introduction au HTML

Qu'est-ce que le HTML ?

HTML (HyperText Markup Language) est le langage standard utilisé pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises.

Le HTML n'est pas un langage de programmation, mais un langage de balisage qui définit la structure de votre contenu.

Structure de base d'une page HTML

<!-- Déclaration du type de document -->
<!DOCTYPE html>
<!-- Élément racine -->
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Titre de la page</title>
</head>
<body>
<!-- Contenu visible de la page -->
  <h1>Mon Premier Titre</h1>
  <p>Mon premier paragraphe.</p>
</body>
</html>

Les éléments et les balises

Les éléments HTML sont les blocs de construction d'une page web. Ils sont définis par des balises :

  • Balise ouvrante : <nom-balise>
  • Contenu : Le contenu de l'élément
  • Balise fermante : </nom-balise>
<tagname>Contenu...</tagname>

Certains éléments n'ont pas de contenu et ne nécessitent pas de balise fermante. On les appelle éléments vides : <br>, <img>, <input>, etc.

Les attributs

Les attributs fournissent des informations supplémentaires sur les éléments :

<a href="https://exemple.com" target="_blank">Visitez notre site</a>

Dans cet exemple, href et target sont des attributs de la balise <a>.

Éléments HTML essentiels

Titres

Les titres sont définis avec les balises <h1> à <h6> :

<h1>Titre Principal</h1>
<h2>Sous-titre</h2>
<h3>Sous-sous-titre</h3>

Utilisez un seul <h1> par page pour le titre principal.

Paragraphes et sauts de ligne

<p>Ceci est un paragraphe.</p>
<p>Ceci est un autre paragraphe.</p>
<p>Ceci est une ligne<br>avec un saut de ligne.</p>

Listes

Il existe deux types de listes :

<!-- Liste non ordonnée -->
<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>

<!-- Liste ordonnée -->
<ol>
  <li>Premier élément</li>
  <li>Deuxième élément</li>
</ol>

Liens

Les liens sont créés avec la balise <a> :

<!-- Lien vers une page externe -->
<a href="https://www.exemple.com">Visitez Exemple.com</a>

<!-- Lien vers une ancre dans la même page -->
<a href="#section2">Aller à la section 2</a>

<!-- Lien pour envoyer un email -->
<a href="mailto:contact@exemple.com">Contactez-nous</a>

Images

Pour insérer une image :

<img src="chemin/vers/image.jpg" alt="Description de l'image" width="500" height="300">

L'attribut alt est obligatoire pour l'accessibilité et le référencement.

Formulaires HTML

Structure de base d'un formulaire

<form action="/traitement.php" method="post">
  <label for="nom">Nom :</label>
  <input type="text" id="nom" name="nom"><br>
  <label for="email">Email :</label>
  <input type="email" id="email" name="email"><br>
  <input type="submit" value="Envoyer">
</form>

Éléments de formulaire courants

  • <input> : Champ de saisie (text, password, email, etc.)
  • <textarea> : Zone de texte multiligne
  • <select> : Liste déroulante
  • <button> : Bouton
  • <label> : Étiquette pour un champ
  • <fieldset> : Groupe de champs
  • <legend> : Titre pour un groupe de champs

HTML5 Sémantique

Les éléments sémantiques améliorent la structure et l'accessibilité :

<header>En-tête de page</header>
<nav>Navigation</nav>
<main>
  <article>
    <section>Section 1</section>
    <section>Section 2</section>
  </article>
  <aside>Contenu complémentaire</aside>
</main>
<footer>Pied de page</footer>

L'utilisation d'éléments sémantiques améliore le référencement et l'accessibilité de votre site.
