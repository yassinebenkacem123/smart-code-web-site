Introduction à React

Qu'est-ce que React ?

React est une bibliothèque JavaScript open-source développée par Facebook pour créer des interfaces utilisateur. Elle permet de construire des applications web complexes à partir de composants réutilisables.

React utilise un DOM virtuel pour optimiser les performances en minimisant les manipulations directes du DOM réel.

Créer une application React

# Créer une nouvelle application avec Create React App
npx create-react-app mon-app
cd mon-app
npm start

Composants et JSX

React utilise des composants comme blocs de construction. Le JSX permet d'écrire du HTML dans JavaScript.

// Composant fonctionnel
function Bienvenue() {
  return <h1>Bonjour, monde !</h1>;
}

// Composant de classe
class Bienvenue extends React.Component {
  render() {
    return <h1>Bonjour, monde !</h1>;
  }
}

Concepts fondamentaux

Props et State

Les props (propriétés) permettent de passer des données aux composants. Le state (état) permet de gérer les données internes d'un composant.

// Utilisation des props
function Salutation(props) {
  return <h1>Bonjour, {props.nom}</h1>;
}
// Utilisation: <Salutation nom="Alice" />

// Utilisation du state avec useState
import { useState } from 'react';

function Compteur() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Vous avez cliqué {count} fois</p>
      <button onClick={() => setCount(count + 1)}>
        Cliquez ici
      </button>
    </div>
  );
}

Cycle de vie et useEffect

Le hook useEffect permet de gérer les effets de bord dans les composants fonctionnels.

import { useState, useEffect } from 'react';

function Exemple() {
  const [data, setData] = useState([]);

  useEffect(() => {
    // Code exécuté après chaque rendu
    fetch('https://api.exemple.com/data')
      .then(response => response.json())
      .then(data => setData(data));

    // Fonction de nettoyage (optionnelle)
    return () => {
      console.log('Nettoyage effectué');
    };
  }, []); // Tableau de dépendances vide = exécuté une fois

  return (
    <ul>
      {data.map(item => (
        <li key={item.id}>{item.name}</li>
      ))}
    </ul>
  );
}

Gestion des événements

La gestion des événements dans React est similaire au DOM, mais avec une syntaxe camelCase.

function BoutonAlerte() {
  function handleClick() {
    alert('Vous avez cliqué !');
  }

  return (
    <button onClick={handleClick}>
      Cliquez-moi
    </button>
  );
}

Gestion d'état avec Context API

Context API permet de partager des données entre composants sans passer par les props.

// Création du contexte
import { createContext, useContext, useState } from 'react';

const ThemeContext = createContext();

function App() {
  const [theme, setTheme] = useState('light');

  return (
    <ThemeContext.Provider value={{ theme, setTheme }}>
      <Toolbar />
    </ThemeContext.Provider>
  );
}

function Toolbar() {
  return <ThemeButton />;
}

function ThemeButton() {
  const { theme, setTheme } = useContext(ThemeContext);

  return (
    <button onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}>
      Changer le thème ({theme})
    </button>
  );
}

Hooks personnalisés

Les hooks personnalisés permettent d'extraire la logique des composants pour la réutiliser.

// Hook personnalisé useWindowSize
import { useState, useEffect } from 'react';

function useWindowSize() {
  const [size, setSize] = useState({
    width: window.innerWidth,
    height: window.innerHeight
  });

  useEffect(() => {
    function handleResize() {
      setSize({
        width: window.innerWidth,
        height: window.innerHeight
      });
    }

    window.addEventListener('resize', handleResize);
    return () => window.removeEventListener('resize', handleResize);
  }, []);

  return size;
}

// Utilisation dans un composant
function DisplayWindowSize() {
  const size = useWindowSize();

  return (
    <p>
      Largeur: {size.width}px, Hauteur: {size.height}px
    </p>
  );
}

Bonnes pratiques

Règles des Hooks

  • N'appelez pas les hooks dans des boucles, des conditions ou des fonctions imbriquées
  • Appelez les hooks uniquement dans des composants React ou des hooks personnalisés
  • Les hooks doivent toujours être appelés dans le même ordre

Optimisation des performances

  • Utilisez React.memo pour mémoïser les composants
  • Utilisez useMemo pour mémoïser des valeurs coûteuses
  • Utilisez useCallback pour mémoïser des fonctions
  • Divisez les gros composants en composants plus petits
  • Utilisez le chargement paresseux avec React.lazy
// Mémoïsation avec React.memo
const MonComposantMemoise = React.memo(function MonComposant({ data }) {
  /* rendu utilisant data */
});

// Mémoïsation de valeurs avec useMemo
const valeurMemoisee = useMemo(() => calculComplexe(a, b), [a, b]);

// Mémoïsation de fonctions avec useCallback
const fonctionMemoisee = useCallback(() => {
  faireQuelqueChose(a, b);
}, [a, b]);

// Chargement paresseux avec React.lazy
const AutreComposant = React.lazy(() => import('./AutreComposant'));
